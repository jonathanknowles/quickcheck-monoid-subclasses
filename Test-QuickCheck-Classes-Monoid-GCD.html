gcdMonoidLaws :: forall a. (Arbitrary a, Show a, Eq a, GCDMonoid a) => Proxy a -> Laws Source #

Laws for instances of GCDMonoid.

Includes the following laws:

Reductivity

isJust (a </> gcd a b)

isJust (b </> gcd a b)

Uniqueness

all isJust
    [ a </> c
    , b </> c
    , c </> gcd a b
    ]
==>
    (c == gcd a b)

Idempotence

gcd a a == a

Identity

gcd mempty a == mempty

gcd a mempty == mempty

Commutativity

gcd a b == gcd b a

Associativity

gcd (gcd a b) c == gcd a (gcd b c)

Distributivity

gcd (a <> b) (a <> c) == a <> gcd b c

gcd (a <> c) (b <> c) == gcd a b <> c

Equivalences

gcd a b == commonPrefix a b

gcd a b == commonSuffix a b

Note that the following superclass laws are not included:

  • monoidLaws
  • commutativeLaws
  • reductiveLaws
  • leftGCDMonoidLaws
  • rightGCDMonoidLaws
  • overlappingGCDMonoidLaws
